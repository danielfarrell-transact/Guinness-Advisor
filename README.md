# Guinness Advisor

![App Screenshots](/screenshots.png)

## Technologies
- SwiftUI
- MapKit
- Firebase Firestore

## :white_check_mark: App Features
- Map View with Custom Annotations for Pubs
- Pub Detail Sheet View
- Pub Dropdown List View
- Pub Info Card with Animation
- Dark Mode Support
